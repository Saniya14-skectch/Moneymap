# MoneyMap – Personal Finance Tracker

Track your money. Understand your habits. Take control.

## 🧾 Overview
MoneyMap is a web-based personal finance application designed to help users track their income, expenses, and savings in one place. The app provides insights into spending behaviour, helps users plan ahead, and encourages better financial decision-making through visual analytics and goal tracking.

## 🚀 Core Features

### 🔐 Authentication
- User Sign Up & Login system
- Secure account access
- Forgot Password functionality (planned)

### 💰 Expense & Income Tracking
- Add, edit, and delete expenses
- Categorise spending (e.g. groceries, rent, shopping, entertainment)
- Track income sources
- View total balance, spending, and savings

### 📊 Spending Insights
- Visual breakdown of expenses using graphs/charts
- Category-based analysis (clothes, groceries, bills, etc.)
- Identify spending patterns over time

### 📅 Financial Planning Tools
- Calendar view to track:
  - Paycheck dates
  - Bill due dates
- Helps users stay organised and avoid missed payments

### 🧮 Built-in Calculator
- Simple calculator for quick financial calculations within the app

### 🎯 Goal Tracker & Rewards System
- Set financial goals (e.g. save $500/month)
- Track progress towards goals
- Reward-based motivation system for achieving milestones

### 🔗 Bank Integration (Future Scope)
- Option to link bank accounts for automatic expense tracking
- View transactions in real-time
- Simplify bill payments directly through the app

## 🌟 Unique Feature (Differentiator)

### 🧠 Smart Financial Awareness System
MoneyMap will include a unique feature that:
- Analyses user spending behaviour
- Provides personalised insights and suggestions
- Detects unusual spending patterns
- Encourages healthier financial habits through subtle nudges

(This feature focuses on behavioural awareness rather than just tracking data, making the app more intelligent and user-centric.)
